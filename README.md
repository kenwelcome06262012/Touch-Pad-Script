# Cisco TMS Touch Panel Inventory Script

This PowerShell script:
- Reads a Cisco TMS export (Excel)
- Connects to endpoints via HTTPS API
- Retrieves connected touch panel information
- Outputs results to CSV/Excel

## Requirements
- PowerShell 7+
- ImportExcel module
- Network access to endpoints

## Usage
1. Export system list from Cisco TMS
2. Update file path in script
3. Run script in PowerShell
4. Review output file
